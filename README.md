## 断点调试
目前搜索没有发现esp8266 可以用PIO 断点调试的可能；其他的平台比如 stm32 可以用stlink 调试器(一种硬件设备)配合进行断点调试

## 作用
利用MQTT 协议吧温度湿度模块的数据传递给MQTT服务器，然后通过客户端订阅实时得到数据；并推送开关空调指令给MQTT服务器远程控制空调开关

## 深度睡眠测试
需要把D0 （GPIO16） 和 RST 在代码烧写后连起来，然后它就会按照周期进行重启
